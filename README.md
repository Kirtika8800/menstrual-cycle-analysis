# 📊 Menstrual Period Analysis Dashboard

## 📌 Overview

This project focuses on analyzing menstrual cycle patterns using data analytics techniques. The goal is to understand relationships between cycle length, stress levels, sleep patterns, symptoms, and exercise habits.

The analysis was performed using Excel for data preprocessing and Power BI for building an interactive dashboard.

---

## 🎯 Objectives

* Analyze menstrual cycle length and consistency
* Study the relationship between stress levels and sleep hours
* Identify the most frequent symptoms during cycles
* Understand the impact of exercise on stress levels
* Build an interactive dashboard for better insights

---

## 📂 Dataset

The dataset contains 120 records with the following features:

* Date
* Cycle Number
* Cycle Day
* Sleep Hours
* Stress Level (Low, Medium, High)
* Symptoms (Cramps, Fatigue, Headache, etc.)
* Exercise (Yes/No)

---

## 🛠️ Tools & Technologies

* **Excel** → Data cleaning & pivot analysis
* **Power BI** → Data visualization & dashboard creation

---

## 📊 Dashboard Features

* KPI Cards:

  * Average Sleep Hours
  * Total Cycles

* Visualizations:

  * Cycle Length Analysis
  * Sleep vs Stress Analysis
  * Symptoms Frequency
  * Exercise vs Stress Analysis

* Filters:

  * Month
  * Stress Level

---

## 🔍 Key Insights

* The menstrual cycle length is consistent (~30 days), indicating regular patterns
* Higher stress levels are associated with slightly lower sleep hours
* Medium stress level shows the highest average sleep duration
* Fatigue and cramps are the most commonly reported symptoms
* Exercise shows limited variation in stress levels
